📖 발음 규칙 학습

한국어의 발음 규칙을 이해하고 올바른 발음을 배워보세요

1. 자음 동화 (Consonant Assimilation)

두 개의 자음이 연결될 때, 한 자음이 다른 자음의 특성을 받아 같은 자음이나 유사한 자음으로 변하는 현상입니다.

👂 발음 비교
신발 → 심발
닭 + 소 → 닭쏘
📝 주요 예시
  • 신발 (ㄴ + ㅂ) → 심발 (비음 동화)
  • 닭소 → 닭쏘 (경음화)
  • 읽다 → 일따 (경음화)

2. 비음화 (Nasalization)

종성 구강음(ㄱ, ㄷ, ㅂ)이 모음이나 비음(ㅁ, ㄴ) 앞에 올 때, 그 구강음이 비음으로 변하는 현상입니다.

👂 발음 비교
옥심 → 옹심
박물관 → 방물관
📝 주요 예시
  • 옥심 (ㄱ → ㅇ before ㅁ) → 옹심
  • 박물관 (ㅂ → ㅁ before ㅁ) → 방물관
  • 독립 (ㄱ → ㅇ before ㅇ) → 동닙

3. 경음화 (Tensification)

어떤 자음이 뒤따르는 약자음(ㄱ, ㄷ, ㅂ, ㅅ, ㅈ)을 쌍자음(경음: ㄲ, ㄸ, ㅃ, ㅆ, ㅉ)으로 바꾸는 현상입니다.

👂 발음 비교
닭소 → 닭쏘
낙제 → 낙째
📝 주요 예시
  • 닭소 (ㄱ → 경음) → 닭쏘
  • 낙제 (ㅈ → 경음) → 낙째
  • 읽고 (ㄱ → 경음) → 일꼬

4. 경구개음화 (Palatalization)

ㄷ, ㅌ, ㄴ이 ㅣ, ㅟ 같은 전설 모음 앞에 올 때, ㅈ, ㅊ, ㅣ로 발음되는 현상입니다.

👂 발음 비교
같이 → 가치
미닫이 → 미다치
📝 주요 예시
  • 같이 (ㄷ + ㅣ) → 가치
  • 미닫이 (ㄷ + ㅣ) → 미다치
  • 굳이 (ㄷ + ㅣ) → 구지

5. ㄹ의 동화 (Liquid Assimilation)

종성 ㄹ 뒤에 자음이 올 때, ㄹ이 다음 자음의 영향을 받아 변하거나, 뒤의 ㄴ이 ㄹ의 영향을 받아 변하는 현상입니다.

👂 발음 비교
칼날 → 칼랄
물도 → 물도
📝 주요 예시
  • 칼날 (ㄹ + ㄴ → ㄴ → ㄹ) → 칼랄
  • 할 때 (ㄹ + ㄷ) → 할 때
  • 물도 (ㄹ + ㄷ) → 물도

6. 격음화 (Aspiration)

종성 ㄲ, ㄳ, ㄺ, ㄻ, ㄼ, ㄽ, ㄾ, ㄿ, ㅀ 같은 쌍자음이나 복합자음이 뒤에서 경음이나 격음을 만드는 현상입니다.

👂 발음 비교
있고 → 있꼬
놓다 → 노타
📝 주요 예시
  • 있고 (ㄲ + ㄱ → 경음) → 있꼬
  • 놓다 (ㄺ + ㄷ → 격음) → 노타
  • 닭도 (ㄱ + ㄷ → 격음) → 닥도

10문제

두 자음이 만나 '어… 나 닮을게' 하고 비슷해지는 현상. 아래 단어의 표기 → 실제 발음을 맞춰보세요!

1. 신문
심문 (ㄴ + ㅂ → 비음 동화)
2. 국민
궁민 (ㄱ + ㅁ → 비음화)
3. 꽃문

4. 학부
항무 (ㄱ + ㅂ → 비음화)
5. 신발
심발 (ㄴ + ㅂ → 비음화)
6. 종로
종노 (ㅇ + ㄴ → 음운 변화)
7. 닭다리
닥따리 (경음화 + 동화)
8. 숙박
숙박 (비음화 없음 - ㅂ 음절)
9. 신랑
실랑 (ㄹ 음운)
10. 백로
뱅노 (ㄱ + ㄴ → 비음화)
