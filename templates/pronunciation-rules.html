📖 발음 규칙 학습

한국어의 발음 규칙을 이해하고 올바른 발음을 배워보세요

1. 자음 동화 (Consonant Assimilation)

두 개의 자음이 만날 때, 한 자음이 다른 자음의 영향을 받아 소리가 바뀌는 현상입니다.

👂 발음 비교
읽다 → 일따
같이 → 가치
📝 예시
  • ㄱ + ㄴ = ㅇㄴ (강남 → 강남, 확인 → 화긍)
  • ㄷ + ㄱ = ㄹㄱ (밭고랕이 → 바꼬랑이)
  • ㄴ + ㅂ = ㅁㅂ (신발 → 심발)

2. 음의 동화 (Vocal Liaison)

모음이나 자음의 발음이 인접한 소리의 영향을 받아 변하는 현상입니다.

👂 발음 비교
그리고 → 그리고
-고 + -요 → -고요
📝 예시
  • 라이 → 래 (모음 변화)
  • 거 + 어 → 거어 (음운 동화)

3. 경음화 (Tensification)

어떤 자음이 뒤따르는 자음을 쌍자음(경음)으로 바꾸는 현상입니다.

👂 발음 비교
닭 → 닭(쌍자음)
국물 → 국뭘
📝 예시
  • 받다 → 받따
  • 낙제 → 낙째
  • 합리 → 합리

4. 비음화 (Nasalization)

구강음(ㄱ, ㄷ, ㅂ) 앞에서 비음(ㅁ, ㄴ, ㅇ)이 따르면, 그 구강음이 비음으로 변하는 현상입니다.

👂 발음 비교
종로 → 종노
닭 → 닭
📝 예시
  • 꼭 → 꼭 (ㄱ → ㅇ before ㄴ)
  • 끝말 → 끝말
  • 약속 → 약속

5. ㄹ의 첨가와 탈락

음절 종성 ㄴ과 모음 ㅡ 사이에서 ㄹ이 첨가되거나, ㄹ이 탈락하는 현상입니다.

👂 발음 비교
늘 → 널
ㄹ (롤) vs (로)
📝 예시
  • 늘어나다 → 널어나다
  • 일어나다 → 일러나다
  • 몸 → 뭄 (ㄴ 음운)

6. 경구개음화 (Palatalization)

ㄷ, ㅌ이 ㅣ, ㅟ 같은 모음 앞에 올 때 ㅈ, ㅊ으로 발음되는 현상입니다.

👂 발음 비교
같이 → 가치
해틀 → 해틀
📝 예시
  • 듣다 → 드따
  • 해뜨다 → 해뜰다
  • 같이 → 가치
