4. 단어 순서 맞추기(Word Order Puzzle)

용도: 어순 연습

기능 아이디어

  • 한국어 문장을 어절 단위 카드로 분리해서 섞어 놓음
  • 예: 오늘 / 친구를 / 만났어요.
  • 사용자가 드래그&드롭으로 순서를 맞추면
    • 정답 체크 → 애니메이션으로 카드가 “착”하고 맞춰지는 연출
  • 상단에는 예쁜 타이머(해가 지고, 달이 뜨는 애니메이션용 타이머)

장점

  • UX적으로 재밌고, 드래그&드롭 연습용으로도 좋음
  • JS 조금만 쓰면 충분히 구현 가능
