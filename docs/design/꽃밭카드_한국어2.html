3. 단어 꽃밭 (Vocabulary Flower Garden)

용도: 단어 암기 + 예쁜 UI 데모

기능 아이디어

  • 화면에 꽃 여러 송이 (카드) 배치
  • 꽃 가운데에 한국어 단어 (예: 학교, 친구, 공부…)
  • 각 꽃을 클릭하면 아래에
    • 뜻 (모국어)
    • 예문 1개
    • 발음(로마자)
  • 배경은 연한 그라디언트 + 초승달/해 아이콘
  • 단어 난이도(색상)
    • 초록: A1
    • 파랑: A2
    • 보라: B1 …처럼 표시

장점

  • 비주얼이 화사해서 “예쁜 UI” 강조 가능
  • 나중에 DB나 API로 단어를 바꾸기도 쉬움
