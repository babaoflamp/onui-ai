“오누이 한국어(ONUI Korean)”은
한국어 서비스라는 정체성과 스토리는 좋지만,
외국인·글로벌 관점에서 발음/인지 측면이 살짝 애매한 네이밍이에요.
조금만 브랜딩을 보완하면 충분히 쓸 수 있는 이름입니다.

1. 의미 & 콘셉트 관점

장점

스토리성이 아주 좋음
옛말 오누이(남매) + 해·달 신화 + “어려움을 이겨내고 빛이 된다”는 서사 →
한국어 교육, 다문화가정, 성장/성취와 잘 맞습니다.
정서적으로 따뜻한 이미지
차갑고 기계적인 AI가 아니라,
“형제자매처럼 같이 공부해주는 AI”라는 느낌을 주기에 딱 좋음.
한국인·다문화 가정 부모에게 감성 어필
‘오누이’라는 단어 자체가 정서적이고 한국적이라
국내 파트너(학교, 지자체, 공공기관, 학부모) 설득용으로는 강점.

단점/리스크

외국인에게는 ‘뜻이 안 보이는 이름’
“ONUI”만 봐서는
언어/교육 서비스인지
한국어인지
앱인지, 게임인지
직관적으로 안 느껴집니다.
다행히 “ONUI Korean”까지 붙이면 어느 정도 해결되긴 합니다.

2. 발음·표기(ONUI) 관점

장점

짧고, 알파벳 4글자라 로고, 앱 아이콘, 도메인에 쓰기 좋음.
“ONUI = 오누이”로 한글-영문 연결이 분명해서
브랜드 스토리 설명할 때 일관성이 있습니다.

우려되는 점


외국인에게 발음이 애매
ONUI → 영어권 기준으로는 오누이, 오누아이, 오뉴이 등 헷갈릴 수 있음
특히 ui 발음은 한국어 배우는 입장에서도 *“의”*라서 난이도가 있는 편.
듣고 적기(hear → spell)
“What app do you use?”
“I use ONUI Korean.”
들은 사람이 Onwi, Onue, Onuy처럼 잘못 쓸 가능성 있음.
해결책:
브랜드 가이드에서 아예 발음을 같이 써주는 전략
예) ONUI Korean (오누이 한국어, 발음: 오-누-이)
영상·랜딩 페이지에서 “오누이”를 충분히 들려주는 브랜딩 필요.

3. 타깃별 적합성

1) 한국어 시험 준비 외국인 (TOPIK 등)

원하는 이미지는:
“신뢰, 체계적, 점수 잘 나오는 서비스”
“오누이”는 감성·따뜻함은 강하지만, ‘시험 전문’ 이미지는 약함
그래서 네임만으로는 “시험 대비 서비스” 느낌이 바로 오진 않음.
보완 전략:
서브브랜드/슬로건으로 보강
예) ONUI Korean Test, ONUI Korean TOPIK Lab,
슬로건: “Your AI tutor for TOPIK success” 같이 명확하게.

2) 발음 교정 외국인

브랜드 이름 자체가 **발음 연습 대상(오누이/ONUI)**이 될 수도 있어서
오히려 재미 요소로 쓸 수 있음.
“오누이처럼 곁에서 발음을 잡아주는 AI 튜터”라는 메시지는 잘 맞음.

3) 다문화가정 (부모·아이 모두)

여기엔 매우 잘 맞는 네이밍
가족, 남매, 따뜻함, 보호·동행 이미지 → 다문화 가정 브랜딩과 궁합 좋음.
다만 부모가 한국어 원어민이 아닐 수도 있어서
“ONUI”가 무엇인지 처음엔 설명이 필요.

4. 비즈니스 관점에서 종합 평가

장점 요약

스토리/세계관 구성에 최적 (해·달, 남매, 성장, 응원 등).
따뜻한 정서 + 한국적인 정체성을 강조하기 좋음.
다문화·가족·어린이/청소년 타깃에는 매우 설득력 있음.
“ONUI Korean” 구조 자체는 글로벌 SaaS 네이밍 포맷과도 크게 충돌 없음.

단점 요약

세계 시장 기준으로 발음/철자에서 한 번 더 설명이 필요한 이름.
“시험/전문성/프리미엄 학습” 이미지는 네임만으로는 약함 →
꼭 태그라인/서브브랜드로 보완해야 함.
같은 이름 쓰는 업체(카페, 쇼핑몰, 학원 등)가 있을 수 있으므로
상표/도메인 확보는 필수 확인 필요(법무·TM 체크 권장).

점수로 말하면

스토리·정체성: 9/10
글로벌 발음·인지 용이성: 6/10
타깃 적합성(전체 평균): 7.5/10 정도 느낌입니다.
